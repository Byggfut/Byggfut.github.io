So You Want to Play Guitar?

Awesome! Here are some of the tools you need to start playing

Tools to Start

  • A guitar (electric or acosutic will work)
  • Picks (Any Thickness will do, but between .6 mm and 1mm is recommended)
  • A sturdy chair (please, save yourself the trouble and try to find one that doesn't have arms)
  • A tuner. While a dedicated tuner would be optimal, there are a number of mobile phone apps that, while not perfect, work admirably.
  • (Optional) a capo. This will be particlalrly useful if you would like to sing along as you play guita. We'll go into more detail later.
